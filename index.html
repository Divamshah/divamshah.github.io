Recent Work

During my school career, I designed and developed a sell/buy web application known as "Gator Trader" for San Francisco State University students in my Software Engineering class, where I not only implemented various frame-works like Handlebars and Bootstrap and but also used Nodej.js, Express.js, Postgres for backend services. Our web-application was hosted on Heroku and supported features from registering on-site to searching items through search bar, filtering and sorting items in user dashboard. All these projects have given me deep understanding of Objected Oriented Design, Model View Architecture, and Agile Software Development.

In addition to that, I built a wine search engine using dataset from Kaggle. I implemented various natural language processing algorithms and scikit learn libraries through python and used Machine Learning to predict wine variety based on the user query. I was challenged with various topic modeling algorithms and overcame them by testing, training and learning the model. I also took an unsupervised approach towards the dataset and used various libraries like matplotlib and pyldavis for data visualization.
